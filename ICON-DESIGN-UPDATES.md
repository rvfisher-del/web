## 🎨 Visual Design System

### **Icon Replacement Strategy:**

| Old Element | New Element | Purpose |
|------------|-------------|---------|
| Emoji icons | Gradient bars | Visual accent without clutter |
| Pictograms | Bold typography | Clearer communication |
| Decorative elements | White space | Breathing room, elegance |
| Stock icons | Professional imagery | Quality over quantity |

### **Color Palette for Accents:**
- Gradient: `from-[#FF8C42] to-[#FF7420]`
- Monochrome: White bars on dark backgrounds
- Subtle: Gray separators (1px)

## 📐 Design Rules Now Enforced

### **Icon Usage Matrix:**

| Use Case | Solution | Example |
|----------|----------|---------|
| Section headings | Bold typography + gradient accent | Canadian Standards |
| Feature highlights | UPPERCASE text + hover effects | Sectors grid |
| Geographic markers | Clean bars + imagery | Canada/Chile |
| Navigation | Minimal SVG (functional only) | Mobile menu |
| Values/Benefits | Professional SVG (brand color) | Social Impact |

### **Never Use:**
- ❌ Emoji icons (🚫)
- ❌ Mixed icon styles
- ❌ Decorative pictograms
- ❌ Generic stock icons
- ❌ Colorful "fun" icons

### **Always Prefer:**
- ✅ Bold typography
- ✅ Gradient accents
- ✅ Professional photography
- ✅ Minimal, functional SVGs
- ✅ White space
