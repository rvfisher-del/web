🚀 TAILWIND TECHNIQUES USED:

→ Arbitrary values:     text-[clamp(...)] tracking-[-0.03em]
→ Opacity modifiers:    bg-white/95 from-orange-500/35
→ Group hover:          group group-hover:scale-105
→ Named groups:         group/link group-hover/link:translate-x-1
→ Responsive prefixes:  md:grid-cols-2 lg:grid-cols-3
→ State modifiers:      hover: focus: after: before:
→ Gradients:            bg-gradient-to-br from- via- to-
→ Transforms:           hover:-translate-y-2 hover:scale-105
→ Transitions:          transition-all duration-300
→ Shadows:              hover:shadow-2xl shadow-[custom]
